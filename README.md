# UTS-Andat-TT
Repository ini berisi dataset yang mengandung kumpulan tweet dari platform X beserta anotasi/label yang menunjukkan apakah setiap tweet berkaitan dengan bunuh diri atau tidak. \\
Dataset ini diambil dari platform Kaggle
https://www.kaggle.com/datasets/aunanya875/suicidal-tweet-detection-dataset/data
